# MC-Tools - tools.mcms.qzz.io


## 1.[末地要塞传送门定位计算器tools.mcms.qzz.io/et/](https://tools.mcms.qzz.io/et/)

本项目是一个专为《我的世界》(Minecraft) 玩家设计的末地要塞坐标计算工具。它基于严谨的两点式直线法算法，帮助你在不开启作弊指令的情况下，通过投掷末影之眼的两个观测点，精准推演出末地要塞的隐藏坐标。

![一张末地传送门的图片](et/index.end.jcd85n.webp)

### 核心功能

● 极简交互：无需安装，打开即用。只需输入两组坐标，一键推演。

● 跨平台：支持 PC 与移动端浏览器(存在些bug)访问，随时随地进行计算。

### 快速开始
如果你希望在本地运行或贡献代码，可以按照以下步骤操作：
#### 1. 克隆仓库

git clone https://github.com/XiaoTC-2026/tools.mcms.qzz.io.git

#### 2. 打开文件
进入项目目录，直接双击 index.html 即可在浏览器中运行。

cd 【你的仓库名】

#### 3. 在线体验
点击仓库主页的 GitHub Pages 链接，直接在浏览器中使用最新版本。
